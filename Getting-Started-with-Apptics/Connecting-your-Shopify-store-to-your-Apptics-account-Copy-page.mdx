---
title: "Connecting your Shopify store to your Apptics account"
---

> 1. In your Apptics Admin panel, click on Stores and select the store you would like to connect.

<Steps>
  <Step title="In your Apptics Admin panel, click on Stores and select the store you would like to connect.">
    ![Apptics 6 Upscaled](/images/apptics6upscaled.png)
  </Step>
  <Step title="In the top Menu select Connect, then click on Shopify.">
    ![Connect S 1 Upscaled](/images/ConnectS1upscaled.png)
  </Step>
  <Step title="Enter your Shop URL from Shopify.">
    ![Connect Apptics 2 Upscaled](/images/connectapptics2upscaled.png)

    NOTE: You will need to use the Shopify URL in this format _yourstore.myshopify.com_.
  </Step>
  <Step title="Go back to Shopify and copy the Admin API access token from the private Shopify app that was explained in the previous chapter">
    ![Capptics 4 Upscaled](/images/capptics4upscaled.png)
  </Step>
  <Step title="Paste it in Apptics under API Token. Also copy and paste API Key and API Secret Key. Click on the Save button.">
    ![Aaptics 5 Upscaled](/images/aaptics5upscaled.png)

    Note: The API Key and API Secret Key fields are not mandatory.
  </Step>
  <Step title="The following notifications will appear in green at the top right, indicating that the integration was successful.">
    ![Capptics 3 Upscaled](/images/capptics3upscaled.png)

    The next step is to sync your products from Shopify, which will also confirm that your store has been successfully connected. You can learn how to do this in the [next chapter](https://docs.apptics.ai/Getting-Started-with-Apptics/Syncing-Products-From-Your-Store-to-Apptics).
  </Step>
</Steps>
