# Initialize checkout session

> 
Initialize a checkout session and automatically redirect to the checkout URL. This endpoint works exactly 
like the current CMS checkout-init page - you can navigate to it with query parameters and it will 
automatically redirect the browser to the checkout page.

Perfect for direct links, email links, or anywhere you want automatic redirection.

**This endpoint is completely public and requires no authentication.**

Example usage:
```
GET /api/public/v1/checkout/init?storeId=store_123&items=[{"variantId":"var_123","quantity":1}]&currency=USD
→ Redirects to: https://checkout.example.com/checkout/ch_abc123/op?token=cms_token_xyz
```
