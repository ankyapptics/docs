---
title: "Initialize checkout session"
api: "POST https://crm.apptics.ai/api/public/v1/checkout/session"
description: "Creates a new checkout session. This generates a unique URL where your customer can complete their purchase."
---

### Headers

<ParamField header="Authorization" type="string" required>
  Enter your API key as: `Bearer <your-api-key>`
</ParamField>

### Body

<ParamField body="amount" type="integer" required>
  The total amount to charge in the smallest currency unit (e.g., `1000` for \$10.00).
</ParamField>

<ParamField body="currency" type="string" required>
  Three-letter ISO currency code (e.g., `USD`, `EUR`).
</ParamField>

<ParamField body="productName" type="string">
  The name of the product or service being purchased.
</ParamField>

<ParamField body="successUrl" type="string" required>
  The URL to redirect the customer to after a successful payment. You can use `{CHECKOUT_SESSION_ID}` as a placeholder.
</ParamField>

<ParamField body="cancelUrl" type="string" required>
  The URL to redirect the customer to if they cancel the checkout process.
</ParamField>

<ParamField body="customerEmail" type="string">
  Pre-fill the customer's email address on the checkout page.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the session was created successfully.
</ResponseField>

<ResponseField name="sessionId" type="string">
  The unique identifier for this checkout session.
</ResponseField>

<ResponseField name="url" type="string">
  The hosted checkout URL. **You must redirect your customer to this URL.**
</ResponseField>

<ResponseExample>

```json Response
{
  "success": true,
  "sessionId": "cs_test_a1b2c3d4e5f6",
  "url": "https://checkout.apptics.ai/pay/cs_test_a1b2c3d4e5f6",
  "expiresAt": "2023-10-27T10:00:00Z"

```

</ResponseExample>
