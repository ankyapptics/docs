---
title: "Initialize checkout session"
api: "GET https://crm.apptics.ai/api/public/v1/checkout/init"
description: "Initialize a checkout session and automatically redirect to the checkout URL."
---

This endpoint works exactly like the current CMS checkout-init page - you can navigate to it with query parameters and it will automatically redirect the browser to the checkout page.

Perfect for direct links, email links, or anywhere you want automatic redirection.

<Note>
  **This endpoint is completely public and requires no authentication.**
</Note>

## Capabilities

### Additional Query Parameters

Any extra query parameters you add will be automatically forwarded to the checkout URL. This is useful for tracking parameters (`utm_source`, `utm_campaign`, etc.) or custom parameters.

### Customer Prefill

You can prefill customer information to streamline checkout:

- `customerId`: Link to existing customer account
- `customerEmail`: Prefill customer email
- `customerFirstName`: Prefill customer first name
- `customerLastName`: Prefill customer last name
- `customerPhone`: Prefill customer phone number
- `customerTags`: Apply tags to customer (comma-separated or JSON array: "segment:vip,source:facebook")

### Checkout Token Forwarding

Include the checkout token in the final checkout URL:

- `includeCheckoutToken`: Set to the desired query parameter name (default: "checkoutToken")
  - Example: `includeCheckoutToken=checkoutToken` will append `?checkoutToken=ch_abc123` to the checkout URL
  - Example: `includeCheckoutToken=token` will append `?token=ch_abc123` to the checkout URL

## Parameters

<ParamField query="storeId" type="string" required>
  The ID of the store initializing the checkout.
</ParamField>

<ParamField query="items" type="string" required>
  JSON string of items to purchase. Example: `[{"variantId":"var_123","quantity":1}]`
</ParamField>

<ParamField query="currency" type="string" required>
  The currency code (e.g., "USD").
</ParamField>

<ParamField query="customerId" type="string">
  Link to an existing customer account ID.
</ParamField>

<ParamField query="customerEmail" type="string">
  Prefill customer email address.
</ParamField>

<ParamField query="customerFirstName" type="string">
  Prefill customer first name.
</ParamField>

<ParamField query="customerLastName" type="string">
  Prefill customer last name.
</ParamField>

<ParamField query="customerPhone" type="string">
  Prefill customer phone number.
</ParamField>

<ParamField query="customerTags" type="string">
  Apply tags to the customer (comma-separated or JSON array).
</ParamField>

<ParamField query="includeCheckoutToken" type="string">
  Set to the desired query parameter name to forward the checkout token.
</ParamField>

<ParamField query="draft" type="string">
  Specific draft ID to load.
</ParamField>

## Examples

<CodeGroup>

```bash
# Basic Redirect
curl -X GET "https://crm.apptics.ai/api/public/v1/checkout/init?storeId=store_123&items=[{\"variantId\":\"var_123\",\"quantity\":1}]&currency=USD"
# Redirects to: https://checkout.apptics.ai/checkout/ch_abc123/op?token=cms_token_xyz
```

</CodeGroup>
