This endpoint works exactly like the current CMS checkout-init page - you can navigate to it with query parameters and it will automatically redirect the browser to the checkout page.

Perfect for direct links, email links, or anywhere you want automatic redirection.

**This endpoint is completely public and requires no authentication.**
