You can use the Emails feature in Apptics to create templates—for example, for abandoned checkout emails—so there’s no need to use Klaviyo or other third-party services.

There are several types of Emails in Apptics:

- Abandoned Checkout
- Customer Promotion Code
- Subscription Rebill Incoming
- Club membership started
- Club membership deactivated
