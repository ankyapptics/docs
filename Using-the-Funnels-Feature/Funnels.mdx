Funnels feature on Apptics allows you to generate a landing page or offer page. To use this feature, you’ll first need to create an offer (or multiple offers). You can find instructions on how to create an offer [here](/knowledge-center/create20an20offer20with20a20discount2026cbc8e2991b80e19cd0e390363bba18).

Funnels are generated using an AI prompt, making setup fast and simple.
